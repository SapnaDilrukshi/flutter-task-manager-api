# Flutter Task Manager API

A Flutter Task Manager application built to learn and demonstrate REST API integration and CRUD operations.

## Features

### Task Management

* View Tasks
* Create Tasks
* Update Tasks
* Delete Tasks

### API Integration

* GET Requests
* POST Requests
* PUT Requests
* DELETE Requests

### Flutter Concepts

* FutureBuilder
* Async/Await
* Navigation
* StatefulWidget
* JSON Parsing
* Form Handling
* Error Handling
* Loading States

## Project Structure

